# Oggy's AI Sports Lab

Mobile-ready Streamlit app for NRL AI predictions.

## Streamlit Cloud deployment

Use these settings:

- Repository: `oggys-ai-sports-lab`
- Branch: `main`
- Main file path: `app.py`

## How to use

1. Open the app.
2. Tap **Load Demo Data**.
3. Tap **Run AI Prediction Engine**.
4. View match winner, try scorer markets and Player DNA.

## Includes

- Demo data loader
- AI feature engine
- Player DNA
- Team DNA 2.0
- Match-up intelligence
- Match simulation
- Ensemble predictions
- Try market probabilities
